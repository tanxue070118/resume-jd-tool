# Resume JD Match Tool

Static website for checking resume and job description matching.

## Publish

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`
- Custom domain: `tanxue.qzz.io`

Cloudflare DNS:

- `CNAME tanxue -> <github-username>.github.io`
- Proxy can be enabled after GitHub Pages custom domain is verified.
